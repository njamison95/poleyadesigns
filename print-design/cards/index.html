Type Classifications Infographic Cards

2023

10 Print infographic cards, 5 x 7 in.

In this project, I hope to educate viewers on type classification by pairing each type class with an equivalent area in Team Cherry's 2017 video game Hollow Knight.

On the front side of each card is the name of a type classification along with a description of its history and application. Examples and a sample alphabet that is in a typeface of the given class are also provided.

On the back you will find the name of an area in Hollow Knight that I believe corresponds with the type class on the front. Hollow Knight is a game that emphasizes exploration. To facilitate such activity, its developers created a sprawling world composed of diverse areas ranging from lush gardens to desolate wastelands. I chose Hollow Knight as my analogous topic because this game, like graphic design, holds much meaning in my life.

Just like each area in the world of Hollow Knight, each type class in the world of typography is distinct and rich with history. I hope this project inspires appreciation and understanding for both topics in a way that goes beyond its original purpose.
